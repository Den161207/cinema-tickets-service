# Сервіс купівлі квитків в кіно 🎬

Проект створено в рамках навчальної практики студентів НУБіП України.

## Опис проекту
Консольний додаток для імітації роботи сервісу з продажу квитків у кінотеатр. Програма дозволяє користувачам переглядати доступні сеанси, бронювати місця та керувати особистим кабінетом.

## Технології
* Мова програмування: **Python 3.x**

## Як запустити
1. Склонуйте репозиторій: `git clone [https://github.com/Den161207/cinema-tickets-service.git]`
2. Відкрийте термінал у папці з проектом.
3. Запустіть програму командою: `python main.py`
